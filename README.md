# CodeAlpha_Student_Grading_Tracker

### 📝 Project Overview
The **Student Grade Tracker** is a Java application designed to help teachers manage student scores. The program allows users to input grades for different students including:
* Adding Name and grade achieved by student in the class
* Displaying all student which are added and there Grades
* Displaying all summary and statistics
